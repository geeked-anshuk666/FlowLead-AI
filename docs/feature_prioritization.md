# Feature Prioritization

Phase prioritizations:

- **Phase 1 (MVP):** Next.js file upload, Express API, direct parsing, and single-model AI mapping.
- **Phase 2 (Scalability & fallbacks):** Redis Pub/Sub workers, SSE streaming updates, Gemini + OpenRouter cascading fallbacks.
- **Phase 3 (Enterprise):** Clustered Databases, AWS S3 file streaming, user-specific mapping profile templates.
